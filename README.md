## A fork of Chris Maltby's JavaScript GB(C) emu.
### Features
• Emulates Gameboy ROMs in your web browser! (NW.js to be precise, haha)
• Supports colour! Although it supports colour, it doesn't support GBC.

### Fork Improvements
• Multiple slots. 
• Requires no hosting
• Has a menu
